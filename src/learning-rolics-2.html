УСТАНОВКА PRIMAVERA P6

Подробная видеоинструкция по установке Oracle Primavera P6

Данное видео содержит ответы на большинство вопросов, возникающих при инсталяции программы.


Перед тем как установить Oracle Primavera P6 на свой компьютер убедитесь, что он соответствует следующим требованиям:

  • Процессор Intel либо идентичный с частотой 1 ГГц или выше
  • Не мене 4Гб оперативной памяти 
  • Операционная система от Microsoft (Windows XP, Windows 7)

Так же вам потребуются дополнительное программное обеспечение: 

  • Java 6 runtime environment (JRE) или Software Developer Kit (JDK)
  • Microsoft .NET Framework 4.
