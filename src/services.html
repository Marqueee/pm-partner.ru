стандарт для всех участников проекта, регламентирующий роли в проекте, процессы управления проектами, шаблоны документов по управлению проектами

ПЛАНИРОВАНИЕ РАБОТ

в т.ч. корректирующих. Результатом чего станет целевой календарно-сетевой график проекта («к чему мы стремимся?»)

инструмента, предназначенного для автоматизации проектной деятельности

КОНТРОЛЬ ПРИМЕНЕНИЯ ПРЕДЛОЖЕННЫХ РЕШЕНИЙ

Сбор данных о ходе выполнения проектов. Анализ текущего календарно-сетевого графика проекта («где мы сейчас находимся на самом деле?»). Координирующая деятельность руководителей проектов для соответствия формируемым руководством предприятия бизнес-требованиям – выполняет Офис Управления Проектами
