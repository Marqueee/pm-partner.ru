УПРАВЛЕНИЕ ПРОЕКТАМИ С ИСПОЛЬЗОВАНИЕМ ПО ORACLE PRIMAVERA P6 EPPM
семинар-практикум

Курс дает понимание возможностей и принципов использования информационных систем управления проектами на примере Oracle Primavera P6

Аннотация

В ходе курса последовательно рассматриваются шаги по разработке расписания проекта и контроля хода его выполнения в системе календарного планирования и управления проектами в Oracle Primavera P6 EPPM.

В течение курса слушатели самостоятельно разрабатывают график проекта, проходя этапы планирования, контроля и завершения при управлении проектами в Primavera P6 EPPM.

Цель (результат) обучения

По окончании курса слушатели смогут:

• самостоятельно разрабатывать и отслеживать календарные графики несложных проектов

• просматривать и анализировать данные уже составленных проектов

• формировать отчётность о ходе выполнения задач с помощью инструментов EPPM

Курс дает понимание возможностей и принципов использования информационных систем управления проектами на примере Oracle Primavera P6 EPPM.
